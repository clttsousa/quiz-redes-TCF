# Quiz Redes de Computadores (TCF Telecom)

## Abrir rápido
- Abra `index.html` no navegador.

## PWA / Offline cache (Workbox)
Service Worker funciona **apenas** quando o site é servido via `http://` (não funciona em `file://`).
O jeito mais simples é usar o Vite.

### Rodar com Vite
1) Instale Node.js (LTS)
2) No terminal, dentro desta pasta:
```bash
npm install
npm run dev
```
Pronto: abre em `http://localhost:5173` com cache offline.

## Atalhos
- 1–4 selecionar alternativa
- Enter / N: Próxima (salva e avança)
- P: Limpar
- Esc: Sair
- H: Ajuda
- G: Glossário
