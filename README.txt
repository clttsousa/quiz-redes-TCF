# Quiz • Redes de Computadores — v5 (banco completo + revisão + mini-aula final)

## Como abrir
1) Extraia o ZIP
2) Abra `index.html` no navegador (PC)

## Correções e melhorias
- Banco completo restaurado (todas as perguntas anteriores)
- Resultado mostra acertos e revisão por pergunta:
  - Sua resposta vs correta
  - "Por que a correta é essa" (explicação)
- Mini-aula final (resumo do tema) baseada nas categorias que caíram na sua prova
- Soundpack mais clean (WebAudio com envelope e low-pass), sons de acerto/erro/timeout + clique suave

Obs: Este projeto usa Tailwind via CDN. Se você quiser 100% offline, eu gero uma versão com CSS embutido local.
